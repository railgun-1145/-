# TransGlass

轻量级 Windows 窗口透明度与置顶管理工具。

## 🚀 快速开始

1. **构建**: 
   ```bash
   cargo build --release
   ```
2. **运行**: 
   运行 `target/release/transglass.exe`。
3. **管理**:
   - 程序启动后将显示控制面板，点击关闭按钮将自动隐藏到系统托盘。
   - 左键点击托盘图标重新打开面板。
   - 右键点击托盘图标可进行一键还原或安全退出。

## ⌨️ 默认快捷键

| 功能 | 快捷键 |
| :--- | :--- |
| **增加透明度** | `Alt + Z` |
| **减少透明度** | `Alt + X` |
| **切换窗口置顶** | `Alt + T` |
| **切换鼠标点透** | `Alt + P` |
| **切换笔点透** | `Alt + Shift + P` |
| **还原当前窗口** | `Alt + R` |
| **还原所有窗口** | `Alt + Shift + R` |
| **重载热键配置** | `Alt + Shift + C` |
| **检查软件更新** | `Alt + U` |

## ⚙️ 自定义配置

你可以通过修改程序同级目录下的 `transglass_hotkeys.json` 来自定义热键与鼠标侧按键绑定（适配数位板/侧键笔）。修改后使用“重载热键配置”即可生效。

### 数位板/侧键笔建议

- 如果你的数位板驱动支持把笔的侧键映射为 `XButton1 / XButton2`，本工具可直接识别并执行动作。
- 默认行为：`XButton1` 降低透明度，`XButton2` 增加透明度（可在配置文件里改成 `toggle_click_through`、`toggle_topmost` 等）。

### 点透模式说明

- “鼠标点透 / 笔点透”用于控制透明窗口是否允许对应输入穿透到下层窗口。
- **双开（鼠标 + 笔都勾选）**：使用 `WS_EX_TRANSPARENT`，输入整体穿透，最稳定。
- **只开其一**：通过低级鼠标钩子配合 `SendInput` 转发按下、移动与抬起，使拖拽/画线能作用到下层窗口；滚轮在仅鼠标点透时也会尝试转发。
- 取决于数位板驱动/应用的输入模式，笔事件有可能被系统当作普通鼠标事件；此时“笔点透”与“鼠标点透”的区分可能不生效。笔设备识别使用系统提供的 `ExtraInfo` 启发式，若驱动不标准可能需同时依赖单一“点透”开关。

### 运行权限说明

- 安装版默认以**普通用户**（`asInvoker`）运行，无法可靠操控**以管理员身份运行**的其它程序窗口（Windows UIPI 限制）。若你需要调节此类窗口，请使用以管理员身份启动的便携副本，或接受该限制。

### 钩子与安全软件

- “仅开一种点透”依赖 `WH_MOUSE_LL`。若安装失败，界面会显示警告；此时请检查是否被安全软件拦截，或尝试以兼容模式排除本程序。

## 📦 打包安装

1. 确保已安装 [Inno Setup](https://jrsoftware.org/isinfo.php)。
2. 运行 `cargo build --release`。
3. 使用 Inno Setup 编译 `TransGlass.iss` 即可生成标准安装程序。
